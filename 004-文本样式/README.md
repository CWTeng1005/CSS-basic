## 文本缩进  

- 属性名：text-indent  
- 取值：  
    - 数字 + px  
    - 数字 + em（推荐：1em = 当前标签的font-size的大小）
- eg：4.1  

## 文本水平对齐方式  

- 属性名：text-align  
- 取值：  
    > - left：左对齐  
    > - center：居中对齐  
    > - right：右对齐  
- 注意点：  
    > 如果需要让文本水平居中，text-align属性给文本所在标签（文本的父元素）设置
